{% comment %} "Idle Words" style "infinity" slide deck. Assumes that it has been called from a project page unless a project ID is explicitly specified using the `project` parameter. Accepts a number of parameters that map to inputs in `widget-slide.html` (and a couple more general ones): `hero_position` Position the `hero_image` on the `left` or `right` for all slides. If unset or invalid, use the value specified by the slide itself. `slide_separator` Liquid truthy/falsy parameter indicating whether or not to add a border between slides. `slide_align` Should slide left/right image and content be `top`, `bottom`, or `center` aligned? If unset or invalid, use the value specified by the slide itself. `title_links` Controls how slide titles should be linked. One of `internal` (links to the anchor of the current slide), `external` (links to the stand-alone slide page), or `none` (no link). If unset or invalid, defaults to `external`. `show_dates` Liquid truthy/falsy value indicating whether or not slide dates should be displayed. `collapse_slides` Liquide truthy/falsy value indicating whether or not slide titles and authors should be suppressed if identical between sequential slides. `table_of_contents` Liquid truthy/falsy value indicating whether a table of contents for the deck should be generated. `table_of_contents_hero` Image to use for the table of contents slide. If unset, defaults to the project hero. `table_of_contents_hero_description` Image alt text for the table of contents slide. If unset, use project title. `table_of_contents_hero_caption` Image caption for the table of contents slide.
